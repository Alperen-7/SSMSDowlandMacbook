# SSMSDowlandMacbook

Microsoft SQL Server, Windows işletim sisteminde olduğu gibi doğrudan Macbook (ARM) işletimine kurulamamaktadır. Bunun için farklı çözüm yöntemleri bulunmaktadır. 
Bunlardan bir tanesi bizim depomuzda yer alan çözüm dosyaları üzerinden ilerlemektedir.
Bizim yöntemimiz sanal makine üzerinden Windows işletim sistemini kurarak Microsoft SQL Server kurulumunu gerçekleştirmektir.
